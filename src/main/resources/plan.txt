-- сервер создает новый объект игры, в качестве первого пользователя записывает nickname
<- пользователь перемещается на новый экран, который связан с сервером по id сессии
-> второй пользователь отправляет запрос на сервер, joinGame,nickname,id сессии
<- сервер устанавливает в объект Game в качестве второго пользователя новый nickname
-> пользователи при помощи чата отправляют команды в TCPшку, sendMsg,nickname,message
-- сервер считывает команду и отправляет её в Game
<- Game отправляет TCPшную команду назад пользователю, класс Game на клиенте считывает команду и выполняет действие
